# Wordle Solver

A rudimentary [Wordle](https://www.powerlanguage.co.uk/wordle/) solver built on a Sunday evening

## Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn serve
```

### Compiles and minifies for production

```bash
yarn build
```

### Lints and fixes files

```bash
yarn lint
```

### Deploy

```bash
bash deploy.sh
```
